# Road Accident Severity Prediction Using Machine Learning

A machine learning project that predicts the severity of road accidents using real-world accident data collected from police records. The project focuses on data preprocessing, feature engineering, exploratory data analysis, and model comparison to classify accidents into severity categories such as Fatal and Grievous.

## Project Overview

Road accidents are a major public safety concern. This project aims to analyze accident-related factors and predict accident severity using machine learning techniques. The system processes raw accident data, extracts meaningful features, and trains multiple classification models to identify patterns associated with severe accidents.

## Dataset

The project uses road accident records obtained from police accident reports. The dataset contains information related to accident conditions, vehicle details, environmental factors, and injury severity.

## Output

The trained models classify accidents into severity categories such as:

- Fatal
- Grievous

The performance of multiple machine learning algorithms is compared to identify the most effective model for accident severity prediction.
